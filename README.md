Attendance
==========
[![Build Status](https://travis-ci.org/illinois/attendance.svg?branch=master)](https://travis-ci.org/illinois/attendance)

Card swipe-based attendance app for CS 225 lab sections.

Setup instructions
------------------
```bash
npm install
cp config.js.sample config.js
# Set up config.js
make serve
```

Testing
-------
```
make test
```

Linting
-------
```
make lint
```
