Attendance
==========
[![Build Status](https://travis-ci.org/kevinwang/attendance.svg?branch=master)](https://travis-ci.org/kevinwang/attendance)

Attendance app for CS 225 lab sections.

Setup instructions
------------------
```
npm install
make serve
```

Testing
-------
```
make test
```

Linting
-------
```
make lint
```
